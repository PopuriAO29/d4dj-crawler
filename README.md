# D4DJ-INFO-SERVER

## Getting Started
1. Clone repository
```bash
git clone https://github.com/gangjun06/d4dj-info-server
cd d4dj-info-server
```
3. build D4DJ-Tool
```
cd D4DJ-Tool
dotnet build --configuration Release
```
2. create .env file
```
SERVER_PORT=9096
ASSET_PATH=./assets
TOOL_PATH=/PATH/TO/D4DJ-Tool/EXECUTEABLE/FILE
```
4. Run!
```
go run main.go
```
